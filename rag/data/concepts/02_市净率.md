# 市净率（P/B Ratio）

## 定义
市净率 = 股票价格 / 每股净资产（BVPS），衡量投资者为公司每 1 元净资产支付多少元。

## 公式
$$P/B = \frac{\text{Price per Share}}{\text{Book Value per Share}}$$

## 适用场景
- **银行 / 保险 / 证券**：资产负债表是核心，PB 比 PE 更稳定。
- **重资产行业**：钢铁、化工、地产等。
- **破净股**：PB < 1 意味着市场给出的估值低于账面净资产。

## 注意事项
1. 商誉、无形资产虚高时，账面净资产可能"虚胖"，PB 失真。
2. 互联网 / 软件等轻资产公司 PB 普遍很高，因为其核心价值不在账面资产。
3. 应结合 ROE 看：高 ROE + 低 PB 通常是好的组合（杜邦分析）。

## 与 ROE 的关系
$$P/E = P/B \times \frac{1}{ROE}$$

即「合理 PB ≈ 合理 PE × ROE」。

# 每股收益（EPS）

## 定义
EPS = 归属于普通股股东的净利润 / 加权平均流通股数，衡量每一股股票对应多少利润。

## 公式
$$EPS = \frac{\text{Net Income} - \text{Preferred Dividends}}{\text{Weighted Avg Shares Outstanding}}$$

## 类型
- **基本 EPS（Basic EPS）**：仅用流通普通股
- **稀释 EPS（Diluted EPS）**：考虑可转债、期权、限制性股票等潜在稀释后的 EPS，更保守
- **TTM EPS**：滚动过去 12 个月

## 投资意义
- 是 PE 估值的分母，直接影响估值合理性
- EPS 增速 > 收入增速：意味着公司在提升利润率或降低稀释
- 大量发新股或可转债 → EPS 被稀释，即便净利润涨股东也未必受益

## 与其他指标关系
- **PE = 股价 / EPS**
- **股息率 = 每股股息 / 股价**
- **派息率 = 每股股息 / EPS**（公司把多少利润分给股东）

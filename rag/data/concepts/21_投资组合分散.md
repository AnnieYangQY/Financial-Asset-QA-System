# 投资组合分散化

## 核心原理
"不要把所有鸡蛋放在一个篮子里" — 通过持有相关性低的资产，**降低非系统性风险**。

## 数学基础（马科维茨现代组合理论）
两资产组合方差：
$$\sigma_p^2 = w_1^2 \sigma_1^2 + w_2^2 \sigma_2^2 + 2 w_1 w_2 \rho_{12} \sigma_1 \sigma_2$$

- 当 $\rho_{12} < 1$ 时，组合波动率小于加权平均，**就是分散化收益**
- 当 $\rho_{12} = -1$ 时，理论上可以构造零波动组合

## 分散维度
1. **行业分散**：消费 / 科技 / 金融 / 医药 / 周期 ≥ 5 大类
2. **风格分散**：成长 vs 价值 / 大盘 vs 小盘
3. **地域分散**：A 股 / 港股 / 美股 / 新兴市场
4. **资产类别分散**：股 / 债 / 商品 / 现金 / REITs / 黄金
5. **时间分散**：定投，平滑择时风险

## 多少只股票才够
- 经验：30 只左右后，组合波动率已接近指数
- 但需要"真分散"：30 只都是新能源 ≠ 30 只跨行业
- 个人投资者更优解：买宽基 ETF，单只标的就分散到 300/500 个公司

## 60/40 经典组合
- 60% 股票 + 40% 债券
- 长期年化 7-8%，最大回撤 -25%~-35%
- 2022 年股债同跌一度引发"60/40 已死"讨论，但 2023 后修复

## 风险平价（Risk Parity）
桥水基金推广：按"风险贡献相等"配置而非"金额相等"，让低波资产（债券）通过加杠杆到与股票同等风险水平。

## 注意
- 过度分散 = 平庸化（Diworsification）：100 只股票相当于买指数，不如直接买 ETF
- 高相关性时分散失效：危机中各类资产相关性会瞬间靠近 1

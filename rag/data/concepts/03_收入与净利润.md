# 收入和净利润的区别

## 一句话总结
**收入是"卖了多少钱"，净利润是"赚了多少钱"**。两者是利润表（损益表）从顶到底的两端。

## 利润表瀑布

```
营业收入（Revenue / Top Line）
  - 营业成本（COGS）
= 毛利
  - 销售费用
  - 管理费用
  - 研发费用
= 营业利润（Operating Income）
  ± 其他收益 / 公允价值变动 / 投资收益
  - 财务费用
= 利润总额（Pretax Income）
  - 所得税
= 净利润（Net Income / Bottom Line）
```

## 关键差异

| 维度 | 营业收入 | 净利润 |
|---|---|---|
| 概念 | 销售商品/服务的总流入 | 扣除所有成本费用税后的最终归属 |
| 含义 | 业务规模 | 业务质量 + 财务状况 |
| 受影响项 | 主要是销量 × 单价 | 成本、费用、税收、非经常损益 |
| 操纵空间 | 较小（但有提前确认风险） | 较大（折旧、减值、一次性损益等） |

## 衍生指标
- **毛利率** = 毛利 / 营业收入
- **净利率** = 净利润 / 营业收入
- **归母净利润**：归属于母公司股东的净利润，剔除少数股东权益后才是给股东的"真净利"
- **扣非净利润**：扣除非经常性损益后的净利润，反映主业真实盈利能力

## 投资视角
- 营收高但净利润低 → 利润率被挤压，需查成本、费用、竞争格局
- 营收低增长但净利润高增长 → 可能是降本增效或规模效应释放
- 净利润远高于经营性现金流 → 警惕利润含金量不足

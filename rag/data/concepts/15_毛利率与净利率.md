# 毛利率与净利率

## 毛利率（Gross Margin）
$$\text{毛利率} = \frac{\text{营业收入} - \text{营业成本}}{\text{营业收入}} \times 100\%$$

反映"产品本身赚不赚钱"，是公司**议价能力 / 成本控制**的核心指标。

### 行业基准
- 软件 / SaaS：70%~90%
- 高端消费品（茅台、爱马仕）：60%~90%
- 互联网平台：30%~60%
- 制造业（汽车、家电）：15%~30%
- 大宗 / 零售批发：5%~15%

## 净利率（Net Margin）
$$\text{净利率} = \frac{\text{净利润}}{\text{营业收入}} \times 100\%$$

反映"扣完所有成本费用税后还能剩多少"，是综合盈利能力。

## 毛利率与净利率的差额
两者之差 = 三费率（销售 + 管理 + 研发）+ 财务费用率 + 税率 + 非经常损益

- 毛利率高但净利率低 → 费用偏高（销售、研发投入大，或管理低效）
- 毛利率低但净利率高 → 极致的成本控制（如沃尔玛、Costco）

## 趋势分析的价值
- 毛利率持续下滑 → 警惕产品力衰退、价格战、原材料涨价
- 毛利率突然提升 → 是真实改善还是会计调整？需查注解
- 净利率与同行差距：可能是商业模式 / 渠道效率 / 品牌溢价的差异

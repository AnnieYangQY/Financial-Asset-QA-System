# 利率与货币政策

## 主要利率
- **美联储基准利率（Federal Funds Rate）**：美国银行间隔夜拆借利率目标区间
- **中国 LPR（Loan Prime Rate）**：贷款市场报价利率，由 18 家银行报价加权
- **MLF（中期借贷便利）**：央行向商业银行提供中期资金的利率
- **逆回购利率**：央行对市场进行短期流动性投放的利率

## 加息 vs 降息

### 加息（Tightening）
- **目的**：抑制通胀、防过热
- **影响**：贷款变贵 → 投资 / 消费降温；存款吸引力上升 → 资金从风险资产撤出
- **资产反应**：股票（尤其成长股）承压、债券下跌、美元走强

### 降息（Easing）
- **目的**：刺激经济、防衰退
- **影响**：贷款变便宜 → 投资 / 消费回暖；存款吸引力下降 → 资金涌向风险资产
- **资产反应**：股票（尤其成长股）受益、债券上涨、美元走弱

## 量化宽松（QE） vs 量化紧缩（QT）
- **QE**：央行直接在二级市场购买国债 / MBS，向市场投放基础货币
- **QT**：到期不再续作，让央行资产负债表缩表

## 实际利率
$$\text{实际利率} = \text{名义利率} - \text{通胀率}$$

- 实际利率 < 0 → 持有现金购买力下降 → 利好金价、风险资产
- 实际利率 > 0 → 现金有真实回报 → 风险资产承压

## 国债收益率曲线倒挂
正常情况下：长期 YTM > 短期 YTM。
**倒挂** = 2 年期 YTM > 10 年期 YTM，历史上是经济衰退的强领先指标（但有时滞）。

## 央行的双重目标
- **美联储**：物价稳定（2% 通胀目标）+ 充分就业
- **欧央行 / 日银**：以物价稳定为主
- **中国央行**：币值稳定 + 经济增长 + 就业 + 国际收支平衡（多目标）
